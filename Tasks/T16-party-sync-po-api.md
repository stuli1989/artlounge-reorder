## Files to Modify

### 1. `api/routes/parties.py`

#### `GET /api/parties/unclassified`
Returns list of unclassified parties needing review.

Response:
```json
[
  {"tally_name": "New Customer XYZ", "tally_parent": "Sundry Debtors", "created_at": "..."}
]
```

#### `POST /api/parties/classify`
Classify a single party.

Request body:
```json
{"tally_name": "New Customer XYZ", "channel": "wholesale"}
```

Validate channel against allowed values: supplier, wholesale, online, store, internal, ignore.
Update `parties` table, set `classified_at = NOW()`.
Also update `transactions.channel` for all transactions with this party name.
Return: `{"success": true, "party": "New Customer XYZ", "channel": "wholesale"}`

### 2. `api/routes/sync_status.py`

#### `GET /api/sync/status`
Returns last sync info and freshness.

Response:
```json
{
  "last_sync_completed": "2026-03-11T02:15:00Z",
  "status": "completed",
  "categories_synced": 23,
  "items_synced": 5432,
  "transactions_synced": 156,
  "new_parties_found": 0,
  "freshness": "fresh",
  "unclassified_parties_count": 0
}
```

Freshness logic:
- `"fresh"` — last sync < 24 hours ago
- `"stale"` — last sync 24-48 hours ago
- `"critical"` — last sync > 48 hours ago or no sync ever

SQL for last sync:
```sql
SELECT * FROM sync_log WHERE status = 'completed'
ORDER BY sync_completed DESC LIMIT 1
```

### 3. `api/routes/po.py`

#### `GET /api/brands/{category_name}/po-data`
Returns PO builder data — all SKUs needing reorder with suggested quantities.

Query params:
- `lead_time` (int, default: supplier's default)
- `buffer` (float, default: 1.3)
- `include_warning` (bool, default: true)
- `include_ok` (bool, default: false)

Logic:
1. Fetch sku_metrics for the category
2. Filter by status: always include critical + out_of_stock, conditionally warning + ok
3. For each SKU, recalculate suggested_qty using the provided lead_time and buffer:
   `suggested_qty = total_velocity * lead_time * buffer`
4. Return list with: stock_item_name, current_stock, total_velocity, days_to_stockout, reorder_status, suggested_qty

#### `POST /api/export/po`
Generate and download an Excel purchase order.

Request body:
```json
{
  "category_name": "Speedball",
  "supplier_name": "Speedball Art Products LLC",
  "lead_time": 180,
  "buffer": 1.3,
  "items": [
    {
      "stock_item_name": "Speedball Monalisa Gold Leaf Sealer...",
      "order_qty": 421,
      "current_stock": 18,
      "velocity_per_month": 54.1,
      "days_to_stockout": 7,
      "notes": ""
    }
  ]
}
```

Excel generation using openpyxl:
```
Row 1:  Art Lounge India — Purchase Order
Row 2:  To: {supplier_name}
Row 3:  Date: {today}
Row 4:  Reference: PO-{brand_prefix}-{date}
Row 5:  Lead Time: {lead_time} days | Buffer: {buffer}x
Row 6:  (blank)
Row 7:  Headers: # | Item Name | Qty | Unit | Current Stock | Velocity/Month | Days Left | Notes
Row 8+: Data rows
Last:   Total Items: X | Total Quantity: Y
Footer: Generated by Art Lounge Stock Intelligence System
```

Return: StreamingResponse with content_type `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`

```python
from fastapi.responses import StreamingResponse
import io

# After generating workbook:
buffer = io.BytesIO()
wb.save(buffer)
buffer.seek(0)
return StreamingResponse(
    buffer,
    media_type="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
    headers={"Content-Disposition": f"attachment; filename=PO-{brand}-{date}.xlsx"}
)
```

### 4. `api/routes/suppliers.py`

Supplier CRUD for managing suppliers and their lead times.

#### `GET /api/suppliers`
Returns all suppliers.

Response:
```json
[
  {
    "id": 1,
    "name": "Speedball Art Products",
    "tally_party": "Speedball Art Products, LLC",
    "lead_time_sea": 180,
    "lead_time_air": 30,
    "lead_time_default": 180,
    "currency": "USD",
    "min_order_value": null,
    "typical_order_months": 6,
    "notes": "Orders typically placed in May and November"
  }
]
```

#### `POST /api/suppliers`
Add a new supplier.

Request body:
```json
{
  "name": "Royal Talens",
  "tally_party": "Royal Talens BV",
  "lead_time_sea": 150,
  "lead_time_air": 25,
  "lead_time_default": 150,
  "currency": "EUR",
  "min_order_value": 500,
  "typical_order_months": 4,
  "notes": ""
}
```

Validate: `name` required, `lead_time_default` required.
Return: the created supplier with id.

#### `PUT /api/suppliers/{supplier_id}`
Update an existing supplier's fields.

Request body: same as POST (partial updates allowed — only include fields to change).
Return: updated supplier.

#### `DELETE /api/suppliers/{supplier_id}`
Delete a supplier. Only allowed if no `brand_metrics` rows reference this supplier (to prevent orphaned data).
Return: `{"success": true}` or 400 if supplier is in use.
